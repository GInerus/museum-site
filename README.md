# Виртуальный музей

Сайт проекта «Виртуальный музей», опубликованный через GitHub Pages.

## О проекте

Виртуальный музей — клиент-серверное приложение, разработанное на C#.

Архитектура проекта:

* WPF Client
* ASP.NET Web API
* SQL Server

Приложение предназначено для просмотра информации о музейных экспонатах и управления данными через административный интерфейс.

## Возможности

* Просмотр экспонатов
* Поиск по экспонатам
* Просмотр подробной информации
* Авторизация администратора
* Добавление и редактирование данных
* Работа через Web API

## Технологии

* C#
* WPF
* ASP.NET Web API
* SQL Server
* Git
* GitHub
